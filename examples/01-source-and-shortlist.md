# Source and shortlist a target universe

End-to-end sourcing: build a long list, screen it, and pull outreach contacts
for the winners. Exercises `creating-tables` → `screening-and-shortlisting` →
`finding-contacts`.

## Prompt

```text
Build a list of Nordic pest control providers for Anticimex, 10–75 employees,
specializing in rodent control. Then shortlist the 10 best acquisition targets
and pull CEO/owner contacts for the shortlist.
```

## What it demonstrates

- Creating a company table from a business activity + geography + size filters.
- Screening the long list into a ranked top-10 shortlist with coverage checks.
- Person-level contact lookup scoped to the shortlisted companies.
