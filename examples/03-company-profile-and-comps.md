# Profile a company and benchmark it

Deep-dive one company, then build and enrich a peer set. Exercises
`company-lookup` → `creating-tables` → `table-enrichment`.

## Prompt

```text
Pull the full Grasp profile for stripe.com — financials, ownership, and
headcount history — then build a table of 15 similar companies and add an
estimated-revenue column.
```

## What it demonstrates

- Single-company profile lookup with expanded sections (financials, ownership, employees).
- Building a similar-company table from a seed company.
- Adding an enrichment column (estimated revenue) to an existing table.
